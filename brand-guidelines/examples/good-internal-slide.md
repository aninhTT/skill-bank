## Off-Brand (Bad)

### Bad Title Slide
```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│              Q4 2025 Results!!!                               │
│              Arial Bold, 36pt, centered                      │
│                                                              │
│              [Small centered image]                           │
│                                                              │
│              Created by the Strategy Team                     │
│              October 2025                                     │
│                                                              │
│                                                              │
│                                   (no footer, no bottom line)│
└──────────────────────────────────────────────────────────────┘
```

**What's wrong**:
- **Wrong font**: Arial instead of Montserrat
- **Centered layout**: Should be left-aligned title with right photo (per Template slide 1)
- **No full-height photo**: Small centered image instead of 40% right photo
- **Too small**: 36pt instead of 57pt title
- **Missing elements**: No Tack logo, no footer, no bottom line, no metadata tags
- **Exclamation marks**: "Results!!!" — off-brand hype
- **Built from scratch**: Not starting from the base template

### Bad Section Divider
```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│              Results                                          │
│              (no eyebrow, no ALL CAPS, wrong position)       │
│                                                              │
│              We are REVOLUTIONIZING how we hit targets!!!     │
│              Calibri, 24pt, centered, with drop shadow       │
│                                                              │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**What's wrong**:
- **No eyebrow**: Missing the ALL CAPS TT Blue eyebrow text
- **Wrong font**: Calibri instead of Montserrat
- **Too small**: 24pt instead of 40–42pt
- **Centered**: Should be left-aligned at (0.5", 2.0")
- **Drop shadow**: Explicitly off-brand
- **Aspirational language**: "REVOLUTIONIZING" — Thumbtack is grounded, not lofty
- **Missing consistent elements**: No footer, no bottom line, no logo
