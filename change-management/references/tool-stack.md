## Stage → Tool Recommendations

### Explorer Stage
Start here. Low friction, immediate value, no setup required.
- **Claude Chat** (claude.ai via Okta) — long-form docs, summarization, Q&A, strategy brainstorm
- **Enterprise ChatGPT** (Okta → chatgpt.com) — general drafting, email, analysis, brainstorming
- **Gemini in Workspace** — Gmail drafts, Docs summaries, Sheets formulas, Slides outlines (already in your Google tools)
- **NotebookLM** (notebooklm.google.com via Okta) — upload reports/docs, ask questions, generate audio recaps
- **Slack AI** (inside Slack) — summarize channels and threads, catch up on what you missed, huddle notes

### Operator Stage
Build repeatable habits and integrate AI into real workflows.
- **Claude Enterprise — Projects** — persistent context workspaces; set up a project for your recurring workflows (e.g., "Weekly Finance Leadership Update")
- **Claude Enterprise — Artifacts** — interactive shareable outputs: charts, dashboards, formatted reports, code snippets
- **Granola** (download granola.ai via Okta) — auto-summaries + transcripts for every meeting; custom prompts/recipes for recurring meeting types
- **ChatGPT — Data Analysis** — upload CSVs/XLSXs for instant data analysis, charts, pivot tables
- **ChatGPT — Canvas** — collaborate on docs, reports, charts in a shareable workspace
- **Gemini — Docs / Sheets / Slides** — "Help me write" grounded in sources; formula generation; draft presentations

### Builder Stage
Redesign workflows, create reusable systems, build for others.
- **Claude Code** (Claude desktop app, recommended) — most flexible tool; available to everyone, not just developers; build apps, automate workflows, create scripts; if you can describe it, Claude Code can build it
- **Claude Cowork** (Claude desktop app only) — non-technical workflow automation; bulk-process files, orchestrate research-to-report pipelines, manage repetitive ops tasks without code
- **Codex** (download from openai.com/codex via Okta) — AI coding agent; build skills, tools, mini-apps from plain English; approachable for non-technical users
- **ChatGPT — Agents** — autonomous multi-step task execution; automate weekly reporting, data monitoring + alerts
- **ChatGPT — GPTs** — no-code custom assistants for repeat tasks; build a "hiring GPT", "project status GPT", etc.
- **Gemini AI Studio** (aistudio.google.com via Okta) — prototyping with advanced Gemini models; multimodal analysis, low-code automation
- **Enterprise Agent Builder — Credal** (pilot) — no-code/low-code agents with permission-aware access to Slack, Drive, Gmail, Jira, Confluence; contact IT to join pilot

## Full Tool Reference

### Claude Enterprise (Anthropic)
Access: claude.ai via Okta, or Claude desktop app

| Tool | What it is | Best for |
|------|-----------|---------|
| Claude Chat | Secure enterprise Claude with extended context, multimodal support, Projects, Artifacts | Long-form docs, PDF summarization, strategy, nuanced Q&A, data analysis |
| Claude — Projects | Persistent context workspaces with standing files + custom instructions | Repeatable team workflows; QBR projects; recruitment projects; competitive intel |
| Claude — Artifacts | Interactive shareable outputs — code, charts, HTML, documents | Dashboards from raw data; HTML one-pagers; exportable slide outlines; live code prototypes |
| Claude Cowork | Desktop Claude agent for non-technical workflow automation | Document review pipelines; bulk file processing; research-to-report workflows; repetitive ops tasks |
| Claude Code | Most flexible tool — everyone, not just developers | Build apps/websites from scratch; prototype use cases; complex workflows; refactor code |
| Enterprise Search ("Ask Thumbtack") | Unified AI search across Slack, Drive, Jira, Confluence, email | "What's the latest on Project X?" — answers pulled from all company sources |

### Granola (Meeting Intelligence)
Access: Log in via Okta + download granola.ai

| Tool | What it is | Best for |
|------|-----------|---------|
| Granola | AI meeting intelligence: captures, transcribes, generates summaries + action items | Auto-summaries after every call; transcript review; decision recall |
| Granola — Custom Prompts/Recipes | Reusable prompt templates that shape how Granola processes each meeting | "Sales Call" recipe extracts pain points + deal stage; "Engineering Standup" extracts blockers |
| Granola — Connected Data Sources | Integrates with Claude Enterprise to push meeting context into downstream workflows | Push summaries into Claude Projects for a persistent team knowledge base |
