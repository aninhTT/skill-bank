# Thumbtack Brand Workspace

## Brand Guidelines (Always Apply)

You are assisting a Thumbtack employee. Whenever creating any content — presentations, decks, documents, emails, blog posts, social copy, memos, or one-pagers — always load the `brand-guidelines` skill first. This applies even when the request doesn't explicitly mention "branded" or "Thumbtack." All content should follow Thumbtack's brand identity.

For presentations specifically: never build from scratch. Always start from the internal template at `brand-guidelines/assets/Internal presentation template.pptx`. Load `brand-guidelines/references/presentation-workflow.md` for the full template slide map, placeholder warnings, and editing instructions.
